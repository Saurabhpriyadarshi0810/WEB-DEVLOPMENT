what are semantic tags?

semantic tags are HTML tags that have a meaning.

semantic tags are used to give meaning to the content inside them.

for example, the <header> tag is used to define the header of a document or a section.

other examples of semantic tags are: <footer>, <article>, <section>, <nav>, and <aside>.
